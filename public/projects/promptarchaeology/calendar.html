Prompt Archaeology

Every Claude Code session logs each prompt. These calendars show when and how intensely each project was built — extracted from 1,224 sessions across 50+ projects, Dec 2025 through Mar 2026.

Note: Prompts extracted from two storage formats: per-session JSONL transcripts (newer) and global history.jsonl (older). Both formats are preserved. Counts exclude tool results, plan implementations, slash commands, and continuation summaries — only real human-typed prompts. Some overlap between formats means totals are approximate.
