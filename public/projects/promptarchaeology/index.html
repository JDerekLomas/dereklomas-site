Every prompt I've written since November 2025—extracted, timestamped, clustered by project. The raw material of building with AI.

Three questions drove this:

  • What's the shape of the work over time?
  • What's the rhythm of context-switching?
  • What patterns actually work?

Methodology

Data extracted from ~/.claude/projects and ~/.claude/history.jsonl using custom Python scripts.

  • 370 full conversation sessions parsed
  • 53 projects identified from directory paths
  • 68 active days (Nov 25, 2025 – Feb 4, 2026)
  • Timestamps used to calculate switching patterns
  • No prompt content exposed—only counts and project names

Built with D3.js for visualizations and Python for data extraction.
